# Remote Cam Preview

在另一台手机上预览拍照画面。

你是否在帮别人拍照时，陷入 拍照-看一眼-调整-拍照-看一眼-调整 的低效循环？那你就很适合用这款app，它可以在拍出来之前就让模特看到成片效果，及时调整。
